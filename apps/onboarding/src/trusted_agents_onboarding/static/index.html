Hermes onboarding demo

Start je eerste Hermes-agent.

Deze demo laat de eerste Mollie-stap zien voor Trusted Agents op Hermes: intakegegevens invullen, tenant aanmaken, Telegram-token veilig opslaan, checkout-link genereren en daarna starten met ChatGPT via het bestaande account of met OpenRouter/API.

1

Intake
Naam, e-mail, plan en Telegram-token worden gecontroleerd.

2

Modelroute
Standaard ChatGPT via OAuth; OpenRouter of een andere API-route kan apart worden afgesproken.

3

Checkout
De backend maakt een Mollie-checkout aan voor de gekozen planprijs.

Onboardingformulier

Standaard logt de klant tijdens installatie zelf in met het bestaande ChatGPT-account via OAuth. Voor OpenRouter of een andere API-route vragen we alleen een key wanneer dat echt nodig is.

Demo-only: gebruik hier geen echte Telegram-tokens. Start standaard met ChatGPT via OAuth; OpenRouter of een andere API-route kan na intake. Voor echte onboarding voegen we Mollie testmode, HTTPS en admin-auth toe.
